# NABHYA – AI Powered Satellite Cloud Removal System

## Overview

NABHYA is an AI-powered geospatial intelligence platform designed to remove cloud contamination from satellite imagery and generate actionable insights for decision-making.

The system leverages Artificial Intelligence, cloud removal techniques, object detection, and smart reporting to transform obscured satellite images into useful geospatial information.

---

## Problem Statement

Satellite images are often affected by cloud cover, making it difficult to obtain accurate ground information. This creates challenges in disaster management, agriculture monitoring, infrastructure assessment, and environmental analysis.

NABHYA addresses this challenge by reconstructing cloud-covered regions and extracting meaningful information from satellite imagery.
